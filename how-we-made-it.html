How We Made It — Silver Jubilee Invitation 03

Technical Craft

Every detail was hand-coded. No UI builders, no Webflow, no WordPress. Pure HTML, CSS, and JavaScript — fast, free, and permanent.

CSS Scroll Animations
IntersectionObserver triggers reveal-up, reveal-left, reveal-right on every section as you scroll down.
Background Music (Saathiya)
Web Audio API with animated equalizer bars. Mobile-safe: plays on first user gesture (tap, scroll, keydown).
RSVP → Google Sheets
Form submits via fetch POST to Google Apps Script web app. Data saved permanently with timestamp, name, attendance, guest count.
Responsive Typography
CSS clamp() for fluid font sizes across all screen widths. No media query hacks — scales smoothly from mobile to desktop.
Animated Leaf Wreath
SVG wreath swaying ±1° on a 6s ease-in-out loop. Sized and centered mathematically to hug the circular photo frame exactly.
Blurred Photo Backgrounds
Same image duplicated as a background layer with filter:blur(40px) + scale(1.3) — fills the circle clip with matching color tones, no white gaps.
GitHub Pages Hosting
Deployed free on GitHub Pages with .nojekyll to bypass Jekyll processing (prevents Liquid template errors from other markdown files).
Live RSVP Dashboard
Admin-only page at /dataofsanya reads all Sheet entries via GET, shows attendance stats, guest totals, and timestamped response table.

  ['click','touchstart','keydown','scroll'].forEach(e =>

How We Made It — Silver Jubilee Invitation 04

The AI Superpower

This project was built using Claude Code with GODSKILL — a compiled set of advanced AI workflow skills. Three skills were central to how we worked.

dispatching-parallel-agents
From GODSKILL — Antigravity Superpower
When asked to create 10 completely different designs simultaneously, we used the parallel dispatch pattern — spinning up 5 independent AI agents in parallel, each responsible for 2 designs. No agent shared context with another. All 10 completed and committed in a single push.

brainstorming
From GODSKILL — Superpowers SDLC
Before writing code, we explored the brief — couple names, occasion, cultural identity (Maharashtrian), song choice, color palette preference, photo availability. This ensured every design decision had intent, not guesswork.
iterative-polish / mobile-first thinking
Applied AI Workflow Pattern
Each feature went through client feedback loops. Mobile layout, photo crop positions, circle sizing, blur intensity — all tuned in real-time via short feedback cycles. "little bit more", "take camera up", "perfect" — each became a precise CSS change, tested, and pushed live instantly.
